## web starter: svelte-kit on Vite with Tailwind

Included libraries/tools:
- [svelte kit](https://github.com/sveltejs/kit/tree/master/documentation) with TypeScript
- [Tailwind CSS](https://tailwindcss.com)
- [eslint](https://eslint.org)
  - [eslint-plugin-svelte3](https://github.com/sveltejs/eslint-plugin-svelte3)
- [prettier](https://prettier.io)
  - [prettier-plugin-sort-imports](https://github.com/trivago/prettier-plugin-sort-imports)

## Running the project

Clone this repo and cd to the directory, then run npm or yarn

```bash
npx degit monsendag/web-starter my-app
cd my-app
npm install 
npm run dev -- --open
```
## build static

```bash
npm run build
```
